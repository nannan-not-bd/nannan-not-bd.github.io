Regularization

Regularization，中文翻译过来可以称为正则化，或者是规范化。什么是规则？闭卷考试中不能查书，这就是规则，一个限制。同理，在这里，规则化就是说给损失函数 加上一些限制，通过这种规则去规范他们再接下来的循环迭代中，不要自我膨胀。

通过分析，我们可以看出，。我们对代价函数增加一个限制条件，限制其较高次的参数大小不能过大。

当越接近0的时候呢，我们的曲线越平滑。

那么为什么更喜欢平滑的曲线呢？

  • If some noises corrupt input when testing —— A smoother function has less influence.
